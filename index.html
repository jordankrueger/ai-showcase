The story

I've worked in progressive politics and nonprofit operations for twenty years. I know ActionKit, Google Workspace, and Freshdesk inside out. I can untangle a broken email pipeline in my sleep. But writing code? That was always someone else's job.

Then I started using Claude Code, and something shifted. I went from "Can you help me write a Python script?" to shipping full-stack web applications in a single sitting. Not toy apps — real tools that I use every day, that my clients pay for, and that hundreds of people depend on.

I'm not a coder/developer, but I want to try and level up my skills via vibecoding with Claude Code.

What surprised me most wasn't the code itself — it was the breadth. In a single day, I might fix a WordPress security vulnerability, build a Stripe-integrated SaaS frontend, set up server monitoring, draft an advocacy newsletter, and configure automated backups. Claude doesn't just write code; it handles the entire surface area of running a technology-dependent life.

I have ADHD, which means I tend to bounce between projects. Most productivity advice says that's a problem. With Claude Code, it became a superpower. I can context-switch between a Firebase app, a Cloudflare Worker, a WordPress plugin, and a Next.js dashboard — and actually ship all four in the same afternoon.

Everything below was built in roughly three weeks. No contractors. No dev team. Just me describing what I needed, and Claude helping me build it.

The stories behind the projects

Zero to Shipped

Performance

10x Wiki Speedup

A 54,000-page wiki went from 1.5 seconds to 0.15 seconds. PHP 8.2 migration, APCu caching, OPcache JIT, and cleaning a 1.38GB stale cache table. The hardest lesson: APCu's CLI and FPM memory pools are completely separate.

Problem Solving

The Backup That Wouldn't Fit

A 40GB archive kept crashing the backup system. The VPS had a 268MB kernel memory cap that killed any tar+gzip operation. After multiple failed approaches, Claude redesigned the entire strategy: separate DB dumps from file sync, no compression.
